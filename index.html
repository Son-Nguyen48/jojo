How to create a pull request

Step by step

* Step one: Create a Repository

- Click new repository

- Enter your name repository

- Scroll down and click Create Repository

* Step two: Open Visual Studio Code and open a Folder

- Create any file for example, i create a html file call index.html

- Then hit Ctrl + J to open terminal

- Then type folow command lines below

+ git init

+ git add .

+ git commit -m "first commit"

+ git branch -M main

+ git remote add origin git@github.com:Son-Nguyen48/create-pull-request.git

+Where is git@github.com:Son-Nguyen48/create-pull-request.git?

+Just copy here when you create your new repository

+ git push -u origin main

* Step three: Create a branch from github website

- Type your branch name that you want to create in here

- Then click create new branch

- Then exit and reopen the Visual Studio Code

- Let take a look the bottom left corner that have a word "main"

- Click it and choose the branch you created before

- Now you switch to the branch you created before

- Just start code something and you'll see something like that

- Click the source control with 3 number like the image

- Then type any message and click Commit

- Then click Sync Changes

- Anh that's it you have created a pull request

- Now Go back the github website
